# 版本管理

如果已经有用.exe安装的，请先卸载掉；再通过nvm安装nodejs, 管理node版本；参考nvm (opens new window)安装对应系统的版本；

ps: win系统安装时一定要设置安装的路径上不能有空格

nvm -h
nvm install 10.18 #nodejs版本v10.18版本；
nvm use 10
nvm alias default 10
nvm ls
